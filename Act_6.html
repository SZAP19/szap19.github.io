Siglas

OSI (Open Systems Interconnection)

Modelo de interconexión de sistemas abiertos, divide la comunicación en 7 capas.

IP (Internet Protocol)

Protocolo para direccionamiento y transmisión de datos en Internet.

TCP (Transmission Control Protocol)

Protocolo que garantiza la entrega ordenada y confiable de datos.

UDP (User Datagram Protocol)

Protocolo rápido pero sin garantía de entrega, usado en transmisiones en tiempo real.

SSH (Secure Shell)

Protocolo seguro para administración remota de servidores.

FTP (File Transfer Protocol)

Protocolo para transferir archivos en redes.

SMTP (Simple Mail Transfer Protocol)

Protocolo para enviar correos electrónicos.

POP (Post Office Protocol)

Servicio para acceder a la oficina postal virtual del correo electrónico.

DHCP (Dynamic Host Configuration Protocol)

Asigna direcciones IP automáticamente en una red.

DNS (Domain Name System)

Convierte nombres de dominio en direcciones IP.

HTTP (HyperText Transfer Protocol)

Protocolo para la comunicación web entre navegadores y servidores.

ARP (Address Resolution Protocol)

Asocia direcciones IP con direcciones MAC en una red local.

MAC (Media Access Control)

Identificador único asignado a dispositivos de red.

LAN (Local Area Network)

Red de comunicación que conecta dispositivos cercanos.
